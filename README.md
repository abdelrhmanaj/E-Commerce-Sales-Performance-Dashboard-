# E-Commerce Sales Performance Dashboard 📊

## Project Overview

This project presents an interactive **Excel dashboard** analyzing e-commerce sales performance.
The dashboard provides insights into revenue, profit, orders, delivery performance, and payment methods.

## Dashboard Preview

<img src="images/excel_dashboard.png" width="900">

## Key Metrics

* Total Net Sales: $1,325,495
* Total Profit: $205,632
* Profit Margin: 6.45%
* Number of Orders: 6,600
* Average Delivery Days: 3.81
* Return Rate: 6.06%

## Dashboard Features

The dashboard includes multiple interactive visualizations:

* **Monthly Sales Trend**
* **Sales vs Profit by Product Category**
* **Sales by Payment Method**
* **Sales by Channel (Web, Mobile App, Marketplace)**
* **Interactive Filters (Region, Channel, Product Subcategory)**

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Transformation

## Dataset

The project uses multiple datasets including:

* orders_2024.csv
* orders_2025.csv
* customers.csv
* products.csv
* returns.csv
* campaigns.csv

These datasets were combined and analyzed to generate business insights.

## Insights

Key observations from the dashboard:

* Mobile App and Web channels generate the highest revenue.
* Certain product categories drive higher profit margins.
* Payment methods show different sales distributions.
* Monthly trends help identify peak sales periods.
